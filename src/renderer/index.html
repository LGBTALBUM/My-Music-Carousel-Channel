Library

資產管理

把音訊、LRC、PV、背景圖或整個專輯資料夾拖放到這裡入庫。 同名檔案會自動配對，例如 song.wav + song.lrc + song.mp4。

拖放檔案或資料夾 支援音訊、LRC、PV、背景圖；專輯建議用下方手動入庫指定 




      

        
手動命名入庫

        
先選擇音訊 / LRC / PV / BG，再手動修改入庫名稱與目標專輯。LRC / PV 會嘗試綁定到同一專輯內同名歌曲；找不到歌曲時會先作為獨立素材入庫。

入庫規則

        

          快速入庫不再把上層資料夾自動當成專輯名稱，會先放入「未分類」。
          需要指定專輯與入庫檔名時，請使用下方「手動命名入庫」。
          同一專輯內同名作品不會重複建立，只會補掛或更新同名 LRC / PV。

Delete

          
刪除管理

        

      

      
這裡可以刪除歌曲記錄，也可以直接刪除 data 內的 LRC、歌曲音訊、PV、背景圖檔案。直接刪除檔案時，系統會同步清理 config.yaml 裡的引用。


      

        
歌曲刪除

        
「只刪記錄」會保留 data 裡的檔案；「刪記錄+未被引用檔案」會同時清掉這首歌專用的音訊、LRC、PV。

/data 檔案刪除

        
可直接刪除 data 內的檔案。刪除音訊會移除使用該音訊的歌曲；刪除 LRC/PV 會清掉對應綁定；刪除 BG 會移除背景圖記錄。
